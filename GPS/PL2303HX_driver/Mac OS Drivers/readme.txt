Release Note, PL2303 Mac OS X 10.6 driver v1.4.0, Prolific Edition
=============================================

Supported device ID:
====================  
    VID:067B
    PID:2303

Kernel extension filename:
====================  
    ProlificUsbSerial.kext

Device base name: 
====================  
    usbserial

Device descriptions: 
====================  
    USB-Serial Controller

Driver version:
====================  
    1.4.0

Installer filename:
====================  
    PL2303_1.4.0.pkg

Installer script:
====================  
    Remove any previous installed driver first.

	1. go to terminal
	2. cd /system/library/Extension
	3. kextunload ProlificUSBSerial.kext
	4. rm -r ProlificUSBSerial.kext

Installer title:
====================  
    Prolific USB to Serial Cable driver for Mac OS X 10.6

 
System Requirement:
====================  
. MacOS 10.6 Snow Leopard (32 and 64 bit kernel)
. USB host controller
. Device using PL-2303H/X
